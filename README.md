Project: Idea Intelligence Engine
Why This Project Exists
I often generate spontaneous ideas — business ideas, scientific thoughts, product concepts.

They are written in notes but never evaluated or researched.

This engine creates a feedback loop:

Idea → Research → Evaluation → Storage → Learning

What It Does
Accepts raw ideas via CLI
Generates optimized search queries
Performs web research (DuckDuckGo)
Deduplicates research results
Evaluates idea brutally but constructively
Repairs malformed JSON automatically
Stores structured data in Spark Vault
Architecture
User Input → Query Generation (LLM) → Web Research → Evaluation (LLM) → JSON Repair → Structured Storage

Design Principles
Modular structure
Config-driven behavior
Cost-conscious architecture
Brutal honesty in evaluation
Separation of research and reasoning
No unnecessary complexity
Current Version
V2 – Research-Integrated Evaluation Engine

Capabilities:

5 query generation
Fast / Balanced / Deep research modes
Deduplication
Structured evaluation
Spark Vault storage
